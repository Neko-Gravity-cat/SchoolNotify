School Notify

A web crawler which grabs announcements from school website.

Why School Notify?

School Notify is a web crawler, it will send emails containing school announcements so that you don't have to do it yourself. 

How can I use it?

There's no way to subscribe to this service online for now, please contact me or clone this project and host it yourself.

Important thing you need to know

School Notify project is build for a specific school only. If you don't have any idea why you're here, we're sorry that this place may not be able to help you.
